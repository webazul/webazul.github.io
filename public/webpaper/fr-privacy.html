Politique de Confidentialité

Dernière mise à jour : 28 avril 2026

Résumé : WebPaper traite vos documents localement sur votre appareil. Nous collectons un minimum de données nécessaires à la fourniture du service. Nous ne vendons pas vos informations personnelles.

1. Qui sommes-nous

WebPaper (« nous », « notre ») est une application mobile développée par WebAzul, exploitée par Jhony Mendonca, basée au Portugal. Cette politique s'applique à l'application WebPaper disponible sur iOS et Android.

Contact : jhony@webazul.pt

2. Ce que nous collectons

2.1 Informations de compte

Lorsque vous créez un compte, nous collectons :

  • Adresse e-mail
  • Nom d'affichage (facultatif)
  • Préférence de langue

2.2 Documents et numérisations

  • Utilisateurs gratuits : Tous les documents sont stockés localement sur votre appareil uniquement. Nous n'avons pas accès à vos documents.
  • Utilisateurs premium : Les documents sont sauvegardés dans le stockage cloud (Google Cloud Storage) à des fins de synchronisation et de sauvegarde. Les documents sont stockés de manière sécurisée et ne sont accessibles que par vous.

2.3 Signatures

Vos images de signature et de paraphe sont stockées dans votre profil utilisateur à des fins de signature de documents. Les signatures des utilisateurs premium sont stockées dans le cloud ; les signatures des utilisateurs gratuits restent sur l'appareil.

2.4 Informations d'achat

Les achats d'abonnement sont traités par Apple (App Store) ou Google (Play Store). Nous recevons une confirmation d'achat et un identifiant de transaction, mais nous ne traitons ni ne stockons les données de carte de paiement.

2.5 Informations sur l'appareil

Nous collectons :

  • Jeton de notification push (si les notifications sont activées)
  • Paramètre de langue de l'appareil
  • Horodatage de la dernière connexion

2.6 Données publicitaires

Pour les utilisateurs gratuits, nous affichons des publicités via Google AdMob. Google peut collecter des identifiants d'appareil et des données d'utilisation pour la personnalisation des annonces. Vous pouvez refuser les annonces personnalisées dans les paramètres de votre appareil. Voir la Politique de confidentialité de Google.

3. Comment nous utilisons vos données

  • Pour fournir des services de numérisation, de signature et de stockage de documents
  • Pour synchroniser les documents entre les appareils (utilisateurs premium)
  • Pour traiter et valider les abonnements
  • Pour envoyer des notifications push (avec votre consentement)
  • Pour afficher des publicités (utilisateurs gratuits)

4. Base juridique du traitement (RGPD)

Pour les utilisateurs de l'Espace économique européen (EEE), nous traitons les données sur la base de :

  • Exécution du contrat : Gestion de compte, stockage de documents, services d'abonnement
  • Consentement : Notifications push, publicité personnalisée
  • Intérêt légitime : Sécurité de l'application, prévention de la fraude, amélioration du service

5. Stockage et sécurité des données

  • L'authentification est gérée par Firebase Authentication (Google)
  • Les données cloud sont stockées sur les serveurs de Google Cloud Platform
  • Toutes les transmissions de données utilisent le chiffrement TLS/SSL
  • Les règles de sécurité Firestore limitent l'accès aux données propres des utilisateurs authentifiés
  • Les documents locaux utilisent le système de fichiers natif de l'appareil et sont protégés par le bac à sable du système d'exploitation

6. Partage des données

Nous ne vendons pas vos données personnelles. Nous partageons des données uniquement avec :

  • Google Firebase : Authentification, base de données, stockage cloud
  • Google AdMob : Publicité (utilisateurs gratuits uniquement)
  • Apple / Google : Validation des achats

7. Transferts internationaux

Vos données peuvent être traitées sur des serveurs Google Cloud situés en dehors de votre pays de résidence, y compris aux États-Unis. Ces transferts sont protégés par des Clauses Contractuelles Types (CCT) et les certifications de conformité de Google.

8. Conservation des données

  • Les données de compte sont conservées tant que votre compte est actif
  • Les documents sont conservés jusqu'à ce que vous les supprimiez
  • Lors de la suppression du compte, toutes les données cloud associées sont supprimées dans un délai de 30 jours
  • Les documents stockés localement uniquement restent sur votre appareil jusqu'à ce que vous supprimiez l'application ou les fichiers

9. Vos droits

Tous les utilisateurs

  • Accéder, corriger ou supprimer les données de votre compte
  • Exporter vos documents à tout moment
  • Supprimer votre compte via l'application

Utilisateurs de l'EEE (RGPD)

  • Droit d'accès (Art. 15)
  • Droit de rectification (Art. 16)
  • Droit à l'effacement (Art. 17)
  • Droit à la portabilité des données (Art. 20)
  • Droit d'opposition au traitement (Art. 21)
  • Droit de déposer une plainte auprès de votre Autorité de protection des données locale

Utilisateurs brésiliens (LGPD)

  • Confirmation du traitement des données
  • Accès à vos données
  • Correction des données incomplètes ou inexactes
  • Anonymisation, blocage ou suppression des données non nécessaires
  • Portabilité des données
  • Informations sur les données partagées
  • Droit de révoquer le consentement

Utilisateurs californiens (CCPA)

  • Droit de savoir quelles informations personnelles sont collectées
  • Droit de demander la suppression des informations personnelles
  • Droit de refuser la vente d'informations personnelles (nous ne vendons pas vos données)
  • Droit à la non-discrimination pour l'exercice de vos droits

10. Vie privée des enfants

WebPaper ne s'adresse pas aux enfants de moins de 13 ans (ou de moins de 16 ans dans l'EEE). Nous ne collectons pas sciemment de données auprès d'enfants. Si vous pensez qu'un enfant nous a fourni des données personnelles, contactez-nous pour les supprimer.

11. Modifications de cette politique

Nous pouvons mettre à jour cette politique de temps à autre. Les modifications importantes seront communiquées via l'application ou par e-mail. L'utilisation continue après les modifications constitue une acceptation.
